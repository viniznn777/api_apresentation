API DE VALORANT

apivalorantgame

Api de Valorant (não-oficial) com muitas informações sobre o jogo de FPS da Riot Games

Se quiser ver todas as informações da API

GET:  https://apivalorantgame.herokuapp.com/info

Se quiser ver todos os personagens

GET:  https://apivalorantgame.herokuapp.com/characters

Se quiser ver um personagem específico, digite seu ID em frente "characters".

Atualmente são 21 Personagens!

EX:  https://apivalorantgame.herokuapp.com/characters/1

Se quiser ver todas as armas

GET:  https://apivalorantgame.herokuapp.com/guns

Se quiser ver uma arma específica, digite seu id em frente a subcategoria.

A primeira arma em cada subcategoria começa com o ID = 1

EX:  https://apivalorantgame.herokuapp.com/pistols/1

EX:  https://apivalorantgame.herokuapp.com/submachine/1

Se quiser ver todos os mapas

GET:  https://apivalorantgame.herokuapp.com/maps

Se quiser ver um mapa específico, digite seu ID em frente "tomap"

Atualmente são 8 mapas!

EX:  https://apivalorantgame.herokuapp.com/tomap/1

EX:  https://apivalorantgame.herokuapp.com/tomap/3
